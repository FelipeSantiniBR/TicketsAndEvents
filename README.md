<a id="Instalação"></a>

## :fire: Instalação e Uso

1. Faça um clone :

```sh
  $ git clone https://github.com/FelipeSantiniBR/TicketsAndEvents.git
```

2. Executando a Aplicação:

```sh
  # Instale as dependências
  $ npm install or $ yarn

  # Inicie a aplicação web
  $ npm start

  # Inicie o json-server
  $ npm run server
```
